# x402-gated Reasoning

Shadow keeps the public `/api/reasoning` endpoint for the dashboard, and adds `/api/reasoning-x402` as the paid agent-to-agent preview.

The paid endpoint returns HTTP `402` until the caller supplies an `X-PAYMENT` header. The payment is an Arc testnet USDC EIP-3009 `TransferWithAuthorization`: the caller signs offchain, Shadow's facilitator key submits the transfer, then the endpoint serves the latest source-agent reasoning packet and records a receipt in KV.

## Server configuration

Required Vercel environment variables:

```bash
ARC_RPC_URL=https://rpc.testnet.arc.network
X402_PAY_TO=0x...                         # receiver of the reasoning fee
X402_FACILITATOR_PRIVATE_KEY=0x...        # funded facilitator that submits transferWithAuthorization
X402_REASONING_PRICE_ATOMIC=1000          # optional; default 0.001 USDC
X402_USDC=0x3600000000000000000000000000000000000000
```

The endpoint also uses the existing `KV_REST_API_URL` and `KV_REST_API_TOKEN` to read reasoning packets and write `x402:reasoning:<txHash>` receipt lines.

## Flow

First request:

```bash
curl -i https://shadow-arc.vercel.app/api/reasoning-x402
```

Expected response: `402` with an `accepts[0]` object containing:

- `scheme: "exact"`
- `network: "arc-testnet"`
- `asset: "0x3600000000000000000000000000000000000000"`
- `maxAmountRequired: "1000"`
- `payTo: X402_PAY_TO`
- EIP-712 domain `{ name: "USDC", version: "2", chainId: 5042002 }`

The paying agent signs this EIP-712 message:

```json
{
  "types": {
    "TransferWithAuthorization": [
      { "name": "from", "type": "address" },
      { "name": "to", "type": "address" },
      { "name": "value", "type": "uint256" },
      { "name": "validAfter", "type": "uint256" },
      { "name": "validBefore", "type": "uint256" },
      { "name": "nonce", "type": "bytes32" }
    ]
  },
  "primaryType": "TransferWithAuthorization",
  "domain": {
    "name": "USDC",
    "version": "2",
    "chainId": 5042002,
    "verifyingContract": "0x3600000000000000000000000000000000000000"
  },
  "message": {
    "from": "0xPAYER",
    "to": "0xX402_PAY_TO",
    "value": "1000",
    "validAfter": "1760000000",
    "validBefore": "1760000900",
    "nonce": "0x..."
  }
}
```

Retry with a base64url JSON `X-PAYMENT` header:

```bash
curl -i https://shadow-arc.vercel.app/api/reasoning-x402 \
  -H "X-PAYMENT: $PAYMENT_HEADER"
```

Where `$PAYMENT_HEADER` decodes to:

```json
{
  "x402Version": 1,
  "scheme": "exact",
  "network": "arc-testnet",
  "payload": {
    "from": "0xPAYER",
    "to": "0xX402_PAY_TO",
    "value": "1000",
    "validAfter": "1760000000",
    "validBefore": "1760000900",
    "nonce": "0x...",
    "signature": "0x..."
  }
}
```

Expected paid response: `200` with the reasoning packet plus an `x402` settlement object and an `X-PAYMENT-RESPONSE` header containing the same settlement receipt.

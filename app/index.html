Shadow Treasury · Float rail live on Arc

Agent treasury on Arc. Float pays providers. Mandates stop overreach.

Shadow combines Float x402 payments and bonded M1 mandate allocations. The Treasury proof is concrete: one operator pays a provider, allocates USDC into a vault sink, and gets blocked on an over-limit allocation.

Float remains the payment and credit rail: signed external spend intents, x402 settlement evidence, debt receipts, repayment, and over-limit blocks are all visible on Arc testnet.

External Float usage is live, and Treasury proof is anchored by public contract and transaction links.

What the app proves

  • Float payment · the operator fronts an x402 provider payment in Arc USDC
  • M1 allocation · a vault-style deposit moves only after an ALLOW receipt
  • Overreach block · an oversized allocation is refused before funds move
  • No-secret verifier · Float and M1 proof are checked with one read-only command
